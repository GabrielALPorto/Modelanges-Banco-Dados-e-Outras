# Banco-de-Dados
Repositório criado para estudos da faculdade ou não sobre banco de dados.

## O que esperar desse repositório

**Aqui eu irei postar comandos SQL, especificamente do PostgreSQL, básicos ou avançados.**
**Também haverá imagens de modelos ER, conversão do modelo ER para o modelo lógico (criação das tabelas no banco de dados).**
**Imagens e arquivos mostrando a passagem do modelo ER para o modelo lógico também serão incluídos nessa pasta.**

🗄️ PostgreSQL Connection Test (conectarBancodados.ts)

Um simples script em Node.js para testar a conexão com banco de dados PostgreSQL.
📋 Pré-requisitos

    Node.js instalado

    PostgreSQL instalado e rodando

    Um banco de dados criado

    Um arquivo .env com configurações

⚙️ Funcionalidade

O script:

    Carrega as credenciais do arquivo .env

    Estabelece conexão com o PostgreSQL

    Executa uma query simples (SELECT * FROM cliente)

    Exibe os resultados no console

    Encerra a conexão
